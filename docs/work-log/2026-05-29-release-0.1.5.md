# Release 0.1.5

## Objective

Publish local LLM support added after `v0.1.4`.

## Acceptance checklist

- [x] `translate-cli` and `xtask` package versions are `0.1.5`.
- [x] `Cargo.lock` workspace package versions are `0.1.5`.
- [x] Standard local validation passes.
- [ ] Version bump commit is pushed to `master`.
- [ ] Tag `v0.1.5` is pushed and release workflow completes.
- [ ] Main repo release has all platform archives and `checksums.txt`.
- [ ] Homebrew tap release and Formula point to `0.1.5`.
- [ ] Homebrew install/upgrade path prints `translate-cli 0.1.5`.

## Version scope

Changes since `v0.1.4`:

- Ollama adapter and model detection.
- LM Studio adapter through `lms`, including `--tool lms` alias.
- Setup recommendation updates: runnable Ollama first, then LM Studio, then existing Agent CLIs unless existing config or `TRANSLATE_CLI_TOOL` overrides it.
- Local model config and env overrides for Ollama and LM Studio.
- Wizard guidance for installed local LLM tools without downloaded models.
- README, architecture note, and fake CLI coverage for local LLM support.
