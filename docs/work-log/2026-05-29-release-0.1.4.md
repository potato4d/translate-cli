# Release 0.1.4

## Objective

`v0.1.3` 以降の README 改善、製品名表記統一、ISO 639-1 言語コード解決を含む新しい version を公開する。

## Acceptance checklist

- [x] `translate-cli` と `xtask` の package version を `0.1.4` に更新する。
- [x] `Cargo.lock` の workspace package versions を `0.1.4` に更新する。
- [x] release 前の標準検証を通す。
- [x] `master` に version bump commit を push する。
- [x] `v0.1.4` tag を push し、release workflow を完了させる。
- [x] main repo release に全 platform archives と `checksums.txt` があることを確認する。
- [x] Homebrew tap release と Formula が `0.1.4` を指すことを確認する。
- [x] Homebrew install/upgrade path で `t --version` が `translate-cli 0.1.4` になることを確認する。

## Version scope

Changes since `v0.1.3`:

- README first-time user flow improvements.
- `Translate CLI` product-name casing normalization.
- Current ISO 639-1 two-letter target language whitelist.
- `kr` compatibility alias for Korean.
