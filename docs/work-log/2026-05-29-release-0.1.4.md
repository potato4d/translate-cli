# Release 0.1.4

## Objective

`v0.1.3` 以降の README 改善、製品名表記統一、ISO 639-1 言語コード解決を含む新しい version を公開する。

## Acceptance checklist

- [x] `translate-cli` と `xtask` の package version を `0.1.4` に更新する。
- [x] `Cargo.lock` の workspace package versions を `0.1.4` に更新する。
- [x] release 前の標準検証を通す。
- [ ] `master` に version bump commit を push する。
- [ ] `v0.1.4` tag を push し、release workflow を完了させる。
- [ ] main repo release に全 platform archives と `checksums.txt` があることを確認する。
- [ ] Homebrew tap release と Formula が `0.1.4` を指すことを確認する。
- [ ] Homebrew install/upgrade path で `t --version` が `translate-cli 0.1.4` になることを確認する。

## Version scope

Changes since `v0.1.3`:

- README first-time user flow improvements.
- `Translate CLI` product-name casing normalization.
- Current ISO 639-1 two-letter target language whitelist.
- `kr` compatibility alias for Korean.

## Local validation

- `cargo fmt --check`: passed.
- `cargo test --workspace`: passed.
- `cargo build --release`: passed.
- `./target/release/t --version`: `translate-cli 0.1.4`.
- `cargo run -p xtask -- build-release`: passed and generated the current `darwin-arm64` archive locally.
- `tar -xOf dist/t-darwin-arm64.tar.gz t | strings | rg 'translate-cli 0\.1\.4|Translate CLI setup'`: release binary contains the expected version and setup text.
- `tar -xOf dist/t-darwin-arm64.tar.gz README.md | rg -n 'ISO 639-1|kr|tl'`: release README contains the updated language-code documentation.
- `rg --files dist | sort`: local host archive, staged README, staged binary, and `checksums.txt` generated.
- `rg -n "node|Node|depends_on" dist`: no Formula was generated from the one-platform local archive; staged README mentions that Node.js is not required.
- `git diff --check`: passed.
